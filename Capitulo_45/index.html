Animar elementos infinitamente

Os desafios anteriores cobriram como usar algumas das propriedades de animação e a regra @keyframes. Outra propriedade da animação é animation-iteration-count, que permite controlar quantas vezes você gostaria de repetir a animação. Exemplo:

animation-iteration-count: 3;

Neste caso, a animação vai parar depois de repetir 3 vezes, mas é possível fazer a animação rodar infinitamente definindo esse valor para infinite. Para manter a bola quicando à direita em um loop contínuo, altere a propriedade animation-iteration-count para infinite.
